## 🎯 About The Project

Welcome to an **interactive side-scrolling adventure**! Navigate through a classic Mario-style platformer world to discover an awesome portfolio experience. This isn't your typical portfolio—it's a **living, breathing game** that combines retro nostalgia with modern web technology.

### 🌟 Key Features

- 🎮 **Interactive Game Mechanic** - Navigate through a Mario-inspired side-scrolling world
- 🎨 **Retro Visual Style** - Classic platformer aesthetics with modern React implementation
- 📱 **Responsive Design** - Play on desktop, tablet, or mobile
- ⚡ **Smooth Animations** - Fluid gameplay and transitions
- 🏆 **Progressive Reveal** - Unlock portfolio content as you progress
- 🎭 **Engaging End Sequence** - Fireworks, banners, and call-to-action finale
- 💻 **Modern Tech Stack** - Built with React 19 and modern JavaScript

## 🚀 Getting Started

### Prerequisites

- **Node.js** (v14.0.0 or higher)
- **npm** or **yarn** package manager

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/pratiknator44/pratik-agarwal.git
   cd mario-portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm start
   ```

4. **Open in browser**
   - Navigate to `http://localhost:3000`
   - Press play and begin your adventure! 🎮

### Build for Production

```bash
npm run build
```

The optimized build will be created in the `build/` folder, ready for deployment.
